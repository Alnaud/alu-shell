# Shell - Init Files, Variables and Expansions

| File | Description |
|------|-------------|
| 0-alias | Creates alias ls="rm *" |
| 1-hello_you | Prints hello <username> using $USER |
| 2-path | Adds /action to end of $PATH |
| 3-paths | Counts directories in $PATH |
| 4-global_variables | Lists all global variables |
| 5-local_variables | Lists all local and global variables |
| 6-create_local_variable | Creates local variable BEST=School |
| 7-create_global_variable | Creates global variable BEST=School |
| 8-true_knowledge | Prints $TRUEKNOWLEDGE + 128 |
| 9-divide_and_rule | Prints $POWER / $DIVIDE |
| 10-love_exponent_breath | Prints $BREATH to the power of $LOVE |
| 11-binary_to_decimal | Converts $BINARY from binary to decimal |
| 12-combinations | Prints all 2-letter combos except oo |
| 13-print_float | Prints $NUM as float with 2 decimal places |
| 14-decimal_to_hexadecimal | Converts $DECIMAL to hexadecimal |
